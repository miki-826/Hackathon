# API 利用許可・データ送信方針

## ランタイム（アプリ実行時）

外部AI API・DB・認証は**一切使用しない**。コア体験はブラウザ内で完結する。

| 用途 | 技術 | データ送信 | 課金 | 状態 |
|---|---|---|---|---|
| 手の検出 | MediaPipe Hand Landmarker（@mediapipe/tasks-vision） | なし（推論はブラウザ内） | なし | 採用 |
| wasm/モデル配信 | jsDelivr / Google Storage の静的CDN | モデルファイルのGETのみ。カメラ映像は送らない | なし | 採用 |
| 効果音 | Web Audio 合成 | なし | なし | 採用 |
| BGM | 同梱mp3 | なし | なし | 採用 |
| 永続化 | なし（メモリ内Blob／任意でLocalStorageメタのみ） | なし | なし | 採用 |

カメラフレーム・プレイ写真は **APIへ送信しない**。認定証は端末内Canvasで生成し、保存操作時のみ端末に保存。

## ビルド時のみ（アプリには含めない）

| 用途 | 技術 | 送信 | 課金 | 許可 |
|---|---|---|---|---|
| UIアセット画像生成 | gpt-image-2（OpenAI, quality=medium） | 英語プロンプトのみ送信。ユーザー写真等は送らない | あり（ユーザー承認済み・枚数上限なし） | APPROVED |

`OPENAI_API_KEY` は画像生成スクリプト実行時のみ使用。リポジトリにはコミットしない（`.gitignore` / `.env.example`）。本番アプリ（Vercel）に環境変数は不要。
