# 柔断 JUDAN — UI Visual Brief

## 世界観の一文
夕暮れの和の庭。剣士が自分の庭の切り株を台にして、豆腐や紐など「柔らかいもの」を素材に手刀の修行をする道場。

## ムード / 質感
- 濃い藍 (#11203a) と墨色 (#0c0f17) を基調、和紙の生成り (#efe2c4)、金 (#c8a24a)、朱 (#b5462f)。
- 障子越しの柔らかい逆光、行灯のあたたかい点光、夕暮れのもや。
- 流血・暴力表現なし。素材は少しコミカルでかわいい。

- bg-play.png (1536x1024) プレイ背景: より暗くもや深め。カメラ映像に重ねるための周辺減光フレーム的構図。

- plaque-wood.png (1024x1024, 透過) 木札（焼き印風の縁）。ボタン外装。

## overlay/余白ルール
- 背景は中央に被写体（カメラ映像 or 切り株+素材）が来るため、中央〜下40%は要素を置かない。
- スプライトは中央配置・余白多め・地面影は薄く（切り株上に再配置するため）。
